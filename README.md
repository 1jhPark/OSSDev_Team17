# OSSDev_Team17
 오픈소스소프트웨어 개발 17번팀에서 진행한 팀 프로젝트인, 게임 <Fruit Flappy Bird>의 저장소입니다.
 
# 게임 소개
Fruit Flappy Bird는 원 터치 모바일 게임입니다. 플레이어는 새를 장애물을 피하고, 과일을 먹으며 새를 최대한 멀리 날려 보내는 것이 목표입니다.
터치 한 번에 점프 한 번 입니다. 점프 감각에 적응하고, 새를 최대한 멀리 날려 보세요!
